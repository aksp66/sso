L'authentification unique,
pensée pour la sécurité.

Nexus relie toutes vos applications à une seule identité : OAuth2, OpenID Connect, PKCE, double authentification et journal d'audit complet — prêts à l'emploi.

Démarrer l'intégration Voir la documentation

OAuth2 + PKCE

Flux Authorization Code conforme RFC 6749, PKCE obligatoire pour les clients publics.

Double authentification

TOTP avec QR code, codes de secours, activation obligatoire pour les administrateurs.

Jetons RS256

Signature RS256 4096 bits, rotation automatique des clés tous les 90 jours.

Journal d'audit

Tous les événements de sécurité (connexions, révocations, consentements) sont enregistrés.

Consentement RGPD

Vos utilisateurs voient et approuvent explicitement les permissions accordées à chaque application.

Intégration simple

Découverte OpenID Connect, JWKS, et une documentation complète pour démarrer rapidement.
