Applications connectées

Chaque application tierce qui utilise Nexus pour vous authentifier apparaît dans la section Applications connectées de votre profil, avec :

  • Le nom de l'application
  • Les permissions (scopes) accordées
  • La date depuis laquelle l'accès est actif

Révoquer l'accès d'une application

  1. Allez dans votre profil, section Applications connectées
  2. Cliquez sur Révoquer à côté de l'application concernée
  3. Confirmez — l'accès est immédiatement invalidé, l'application devra demander une nouvelle autorisation
Révoquer une application ne supprime pas votre compte ni vos données dans cette application — cela coupe uniquement sa capacité à s'authentifier via Nexus en votre nom.

Pourquoi le consentement est demandé à nouveau ?

Si une application demande de nouvelles permissions (par exemple l'accès à votre e-mail alors qu'elle n'avait que votre identifiant), Nexus vous présente à nouveau l'écran de consentement pour ces scopes supplémentaires.
