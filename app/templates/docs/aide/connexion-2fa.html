Se connecter & activer la 2FA

Connexion

  1. Rendez-vous sur la page de connexion
  2. Saisissez votre e-mail et votre mot de passe
  3. Si la 2FA est activée sur votre compte, saisissez le code à 6 chiffres de votre application d'authentification
Après 10 tentatives de mot de passe incorrectes, votre compte est verrouillé automatiquement pendant 15 minutes.

Activer la double authentification (2FA)

  1. Depuis votre profil, cliquez sur Activer la 2FA
  2. Scannez le QR code avec une application comme Google Authenticator, Authy ou 1Password
  3. Saisissez le code à 6 chiffres généré pour confirmer l'activation
  4. Conservez précieusement les 10 codes de secours affichés — ils ne seront plus jamais montrés

Codes de secours

Si vous perdez l'accès à votre application d'authentification, utilisez un code de secours à la place du code TOTP sur l'écran de vérification. Chaque code de secours est à usage unique.

Vous n'avez plus de codes de secours valides et avez perdu votre application 2FA ? Contactez votre administrateur, qui peut réinitialiser votre 2FA depuis le panneau d'administration.
