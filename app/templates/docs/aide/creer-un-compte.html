Créer un compte

Vous pouvez créer votre compte Nexus vous-même, sans intervention d'un administrateur, depuis la page Créer un compte (accessible aussi depuis la page d'accueil et la page de connexion).

Étapes de l'inscription

  1. Renseignez un nom d'utilisateur, votre adresse e-mail et un mot de passe (8 caractères minimum)
  2. Un e-mail de confirmation vous est envoyé — le lien est valable 24 heures
  3. Cliquez sur ce lien pour activer votre compte : tant qu'il n'est pas confirmé, la connexion reste bloquée
  4. Connectez-vous, puis nous recommandons d'activer immédiatement la double authentification (2FA)
Un administrateur peut aussi créer un compte pour vous depuis le panneau d'administration — dans ce cas, aucune confirmation d'e-mail n'est nécessaire, le compte est actif immédiatement.
Si vous êtes administrateur, la 2FA est obligatoire pour accéder au panneau d'administration.

Mot de passe oublié dès la première connexion ?

Utilisez le lien Mot de passe oublié sur la page de connexion pour en définir un nouveau par e-mail.
